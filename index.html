My Games

Doomdrop - FPS infinite runner

Created with the Unity engine. What started off as a personal project to learn some basic C# quickly turned into what is Doomdrop today. I used the 4 month development time of Doomdrop to learn about how systems in the Unity engine work, as well as building up new general skills on areas of game design I had never touched on before, such as AI, audio, and animation. While the initial vision for the game was much larger and I had to continually whittle down the scope of the project, I hope that this smaller-scaled project with an emphasis on player movement and gunplay is both a good foundation for the rest of planned game, as well as a good representation of my skills as a game developer! (Current WIP, continually updated!)

Hack & Smash - Platform Fighting Game Demo

Coded in Godot & GDScript. Inspired heavily by Nintendo's Super Smash Bros. Melee, Hack & Smash was a GameJam project that was born from the idea that not all single player modes in fighting games had to be boring, second-thought additions to these titles. With the goal to eventually turn the game into a rogue-lite 3v3 team-fighter, this small Demo showcases the basic combat system of the main game, albiet with only one placeholder character. I do hope to one day return and finish this project, when more time/resources are at my disposal.
